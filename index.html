北海道 — スポット位置

ランドスケープ / イサム・ノグチ

1モエレ沼公園

彫刻家イサム・ノグチが構想した総合公園。ガラスのピラミッド・プレイマウンテンなど彫刻的構造物が広大な敷地に点在。ランドスケープ建築の世界的傑作。

明治洋風建築

2北海道庁旧本庁舎（赤レンガ庁舎）

1888年竣工のアメリカン・バロック様式。レンガ造り明治官庁建築の代表。国指定重要文化財。内部の展示も充実し無料で見学可能。

安藤忠雄 / 宗教建築

3水の教会（星野リゾート トマム）

安藤忠雄設計・1988年竣工。大自然の中に水盤と十字架が浮かぶ幻想的な教会建築。「教会三部作」の一つ。BCS賞受賞。早朝の一般見学も可能。

詳細 →
安藤忠雄 / 霊廟建築

4頭大仏殿（真駒内滝野霊園）

安藤忠雄設計。ラベンダーの丘から頭だけのぞく巨大大仏が印象的。トンネル状のアプローチを抜けると打ち放しコンクリートの円形空間に大仏が現れる。唯一無二の体験。

産業遺産

5小樽運河・石造倉庫群

大正期整備の運河沿いに石造倉庫が連なる近代産業遺産。保存・活用事例として建築的に参考になる。夜景も美しい。

詳細 →
コロニアル様式

6函館・旧函館区公会堂

1910年竣工。青と黄のツートンカラーが印象的なコロニアル様式の明治洋館。道南の歴史的建造物の代表格。函館の街並み全体も建築的見どころ。

安藤忠雄 / リノベーション

7北菓楼 札幌本館

1926年建築の北海道庁立図書館を安藤忠雄がリノベーション（2016年）。歴史的外観を保存しつつ、内部は3層吹き抜けの開放的な空間に。グッドデザイン賞受賞。お菓子も楽しめる。

隈研吾 / リゾート建築

8ワン・ニセコ・リゾート・タワーズ

隈研吾設計のニセコのラグジュアリーホテル。樹皮付きの木材をひさしに用いアイヌ文化の自然観を体現。国際的リゾート開発の現代建築事例として注目。

軽井沢 — スポット位置

現代建築 / 西沢立衛

1軽井沢千住博美術館

西沢立衛設計・2011年。急傾斜地に建つ壁のないガラス張り美術館。自然光・傾斜・自然との融合が唯一無二。プリツカー賞建築家の代表作の一つ。

坂茂 / 話題 / 2019〜2023

2ししいわハウス（SSH No.01・02）

プリツカー賞受賞の坂茂設計・2019/2022年。木々の間を縫う曲線的木造建築。「泊まれる建築作品」として話題を呼ぶブティックホテル。アート作品も館内に多数。

西沢立衛 / 2023年完成

3ししいわハウス（SSH No.03）

西沢立衛設計・2023年完成。黒く塗装した杉板外壁と無垢ヒノキ内装の対比が鮮烈。回廊から望む中庭・自然との緻密な関係性が建築的に非常に興味深い。坂茂棟と徒歩圏に3棟並ぶ軽井沢の建築聖地。

土木遺産 / 明治期

4碓氷第三橋梁（めがね橋）

1893年竣工の4連アーチレンガ橋。高さ31mの明治土木建築の傑作。国指定重要文化財。エンジニアリングの美と歴史的建築の双方から学べる必見スポット。

明治洋館 / 国重文

5旧三笠ホテル

1906年竣工の純洋風木造ホテル。国指定重要文化財。2025年10月に大規模修復完了・内部見学再開。明治の別荘文化と洋風木造建築技術を体感できる。

現代リゾート / 星野

6ハルニレテラス（星野エリア）

湯川沿いの森に点在するパビリオン型商業施設。自然とシームレスに統合されたリゾート開発の現代的成功事例。建築と商業・自然の融合を学べる。

宗教建築 / 明治

7軽井沢ショー記念礼拝堂・聖パウロカトリック教会

軽井沢の避暑地文化の原点・ショー牧師が1895年に建てた礼拝堂。アントニン・レーモンド設計の聖パウロ教会（1935年）も近接。西洋宗教建築の軽井沢における歴史的文脈を学べる。

現代建築 / セゾン

8セゾン現代美術館

広大な庭園に彫刻・現代アートが点在するパビリオン型美術館。屋外展示と建築空間の関係、ランドスケープと芸術の融合を体感できる。マティス・ピカソ等の作品も収蔵。

福岡 — スポット位置

現代建築 / 黒川紀章

1福岡銀行本店

黒川紀章設計・1975年。花崗岩外壁がマッシブに立ち上がり中間部に大きな外部空間を抱く構成が独特。銀行建築の象徴性・堅実性と都市への公共性を両立した名作。天神駅徒歩1分。

国立博物館 / 菊竹清訓

2九州国立博物館

菊竹清訓設計・2005年。波打つダブルスキンガラス外観が周囲の山並みに溶け込む。国産木材を活用した和モダンの大空間。太宰府天満宮に隣接し文化財とともに見学できる。

隈研吾 / 木組み

3スターバックス太宰府天満宮表参道店

隈研吾設計。「自然素材による伝統と現代の融合」をテーマに釘を1本も使わない木組み構造が入口から奥へ続く。伝統技法の現代建築への応用の傑作として世界的に著名。

現代建築 / 磯崎新

4北九州市立美術館

磯崎新設計・1974年。2つの直方体が丘の上に突き出た「丘の上の双眼鏡」。磯崎新のキャリア初期の代表作でプリツカー賞建築家の思考の原点が宿る記念碑的建築。

世界の建築家通り

5百道浜「世界の建築家通り」

1989年アジア太平洋博覧会を契機に開発。マイケル・グレイブス、黒川紀章ら世界7人の建築家が設計した集合住宅・商業施設が1エリアに集中。多様な現代建築を一度に体験可能。

伊東豊雄 / 有機建築

6ぐりんぐりん（アイランドシティ中央公園）

伊東豊雄設計・2005年。湾曲したガラス屋根が丘に溶け込む体験学習施設。雨水循環・自然採光を活かした環境建築の先駆。アール・ヌーヴォー的有機造形の現代解釈として評価が高い。

藤本壮介 / Park-PFI

7明治公園（博多コネクティッド）

藤本壮介がデザイン監修・2025年供用開始。博多駅前の大規模再開発「博多コネクティッド」エリア内に整備。5つの広場と立体回廊を持つ都市型公園。公民連携（Park-PFI）の先進事例として建築・都市計画の観点から注目。

星野リゾート

軽井沢

星野リゾート宿泊夕食

星のや軽井沢・ハレニレの木など星野系列の宿泊夕食は非日常の体験。会社の福利厚生として印象に残る研修になる。

目安：¥15,000〜40,000（宿泊込み）

福岡のグルメ

福岡

博多ラーメン（豚骨）

福岡発祥・細麺とコクのある豚骨スープ。「一蘭」「一風堂」発祥の地で本場の味を体験。屋台でも楽しめる博多の食文化の象徴。

目安：¥800〜1,200

福岡

博多もつ鍋

牛もつをたっぷりの野菜・豆腐と煮込む博多名物。醤油・味噌・塩など各店が独自のスープを展開。大人数でのチームの懇親会に最適。

目安：¥2,500〜4,000

福岡

博多屋台（中洲・天神）

全国でも珍しい屋台文化が今も息づく中洲・天神。串焼き・おでん・ラーメン・明太料理を並んだ屋台で楽しむ非日常体験。

目安：¥2,000〜3,500

福岡

水炊き・水炊き専門店

博多を代表する鍋料理。コラーゲン豊富な鶏ガラスープで鶏肉・野菜を煮込む。旅の締めに格式ある水炊き老舗での宴会は研修旅行の定番。

目安：¥4,000〜8,000

福岡

辛子明太子・明太料理

福岡名物・辛子明太子を使った多彩な料理。「ふくや」「やまや」などの直営店や明太料理専門レストランで本場の味を楽しめる。

目安：¥1,500〜3,000

福岡

太宰府・梅ヶ枝餅

太宰府天満宮参道名物。米粉の外皮で粒あんを包んで焼いた素朴なお餅。建築見学の後に参道をそぞろ歩きしながら味わう定番グルメ。

目安：¥150〜200/個

台湾のグルメ

台湾

小籠包（ショーロンポー）

台湾を代表する点心。薄い皮の中にアツアツのスープと肉あんが詰まった一口サイズ。「鼎泰豊」など有名店での食事は研修旅行の定番ハイライト。

目安：NT$200〜500（¥900〜2,200）

台湾

台湾夜市グルメ

士林・饒河・逢甲など全国各地の夜市で台湾B級グルメを食べ歩き。胡椒餅・臭豆腐・鶏排・タピオカなど多彩な屋台料理は懇親に最適。

目安：NT$100〜300（¥450〜1,350）

台湾

牛肉麺・魯肉飯

台湾ソウルフード。煮込んだ牛すじ肉と濃厚スープの牛肉麺、豚の角煮飯・魯肉飯はどこでも食べられる台湾の定番日常食。

目安：NT$120〜250（¥540〜1,125）

台湾

台湾式火鍋（shabu shabu）

鍋にたっぷりの野菜・肉・海鮮を入れるスタイルが人気。「海底撈」など大型チェーン店も多く、46名の大人数にも対応できる宴会向き。

目安：NT$400〜800（¥1,800〜3,600）

台湾

宮原眼科スイーツ（台中）

日本統治期の眼科診療所をリノベーションした話題のスイーツ店。図書館のような内装でアイスクリームやパイナップルケーキを楽しむ。建築と食の融合。

目安：NT$150〜300（¥675〜1,350）

台湾

台湾茶・茶芸体験

烏龍茶・高山茶など台湾が誇る茶文化。九份・迪化街の茶藝館で本格的な台湾茶を楽しむ体験は研修旅行の文化的ハイライトになる。

目安：NT$200〜500（¥900〜2,250）
